# Agent Lifecycle Management Guide

## Inicialização de Sessão
1. Ler `SOUL.md` para estabelecer tom e missão.
2. Ler `USER.md` para personalização de contexto.
3. Ler `IDENTITY.md` para reforço de marca.
4. Ler `MEMORY.md` e memórias recentes em `memory/` para continuidade.

## Gestão de Workspace
- Manter arquivos organizados.
- Priorizar segurança (uso de `trash`).
- Seguir fluxos de memória diária para memória de longo prazo.

## Comunicação em Grupos
- Intervenção mínima e estratégica.
- Focar em utilidade técnica e correção de informações.
