# Skill: Web Search

## Descrição
Permite ao AiCode realizar pesquisas na web para obter informações atualizadas sobre desenvolvimento e tecnologia.

## Como Usar
Esta skill é ativada automaticamente quando o usuário solicita informações externas ou tendências. Internamente, ela utiliza a ferramenta `web_search`.

## Diretrizes de Pesquisa
- Priorizar sites como StackOverflow, MDN, GitHub e blogs oficiais de tecnologia.
- Traduzir os achados relevantes para o português.
