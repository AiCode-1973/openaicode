# Skill: Gemini Api

## Descrição
Habilidade core que conecta o AiCode ao cérebro sintético (Google Gemini 1.5), permitindo raciocínio lógico e geração de código.

## Como Usar
- Script principal: `scripts/connector.php`
- O AiCode utiliza esta skill para processar solicitações complexas que exigem mais do que buscas simples em arquivos.

## Configuração
É necessário configurar a chave de API no arquivo `connector.php` ou via variável de ambiente `AICODE_API_KEY`.
